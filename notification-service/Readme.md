# Email cases - Auth Service
- Register - Send OTP
- After Successful Registration Send Welcome email
- Login - Notification message
- Forgot Password - Send OTP
- After Successful Forgot Password Change Password
- Change Password - Send Notification message
- Enable TOTP Verification - Send Notification message
- Disable TOTP Verification - Send Notification message
- Enable Email 2FA Verification - Send Notification message
- Disable Email 2FA Verification - Send Notification message
