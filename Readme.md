# ExpenseMate

## API Gateway
- Expose all the traffic to the internet
- Validate all the incoming requests

## Auth Service
- Handle Authentication and Authorization

## Expense Service
- Handle Expense Creation
- Split Logic
- Receipt Storage

## Notification Service
- Handle Notifications
